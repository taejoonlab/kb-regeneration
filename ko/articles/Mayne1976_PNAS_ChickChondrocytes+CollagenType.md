# Changes in type of collagen synthesized as clones of chick chondrocytes grow and eventually lose division capacity

## Citation (NLM)
Mayne R, Vail MS, Mayne PM, Miller EJ. Changes in type of collagen synthesized as clones of chick chondrocytes grow and eventually lose division capacity. Proc Natl Acad Sci USA. 1976;73(5):1674-1678. doi:10.1073/pnas.73.5.1674

**DOI:** [https://doi.org/10.1073/pnas.73.5.1674](https://doi.org/10.1073/pnas.73.5.1674)

---

## Background

배양 조건에서 다양한 조직 기원의 섬유아세포는 제한된 수명을 가지며, 결국 분열 능력을 상실한다. 연골세포와 같은 분화 세포의 in vitro 노화에 대한 연구는 상대적으로 드물다. Chacko et al.(1969)은 닭 배아 연골세포 클론을 계대 배양할수록 다각형(polygonal) 형태의 연골세포에서 편평한 아메바형 세포로 형태 변화가 일어나며, 이들 세포는 metachromatic 기질을 축적하지 못하는 탈분화(dedifferentiation) 현상을 관찰하였다. 한편, 5-bromo-2'-deoxyuridine(BrdUrd) 처리 단기간 후에도 유사한 형태 변화가 관찰되었다. 본 연구의 목적은 연골세포 클론이 in vitro에서 분열 능력을 상실할 때까지 계대 배양하면서 콜라겐 생합성 유형의 변화를 추적하는 것이다.

---

## Key Experiment Methods

1. **연골세포 클론 분리**: 13일령 닭 배아 흉골(sternum)에서 연골세포를 분리하고, 단일 세포를 유리 미세피펫으로 분리하여 60mm 배양 접시에 단일 클론 배양.
2. **계대 배양**: 각 계대에서 세포가 고밀도에 도달한 후 1-4회 계대. 세포 노화 시점까지 7-9주간 배양.
3. **콜라겐 표지 및 분리**: 각 계대 종료 시점에 [2-³H]glycine으로 24시간 표지. 세포층과 배지를 함께 수거하여 25% ammonium sulfate로 콜라겐 침전, 0.5M 초산 추출.
4. **CM-셀룰로오스 크로마토그래피**: 변성된 콜라겐 α 사슬을 분리하여 α1:α2 사슬 비율 계산. 선형 NaCl 구배(0-0.14M)로 용출.
5. **차별 염 침전(differential salt precipitation)**: Type I 콜라겐은 2.4M NaCl에서 침전시키고, Type I trimer는 0.01M Na₂HPO₄에서 침전시켜 분리.
6. **CNBr 절편 분석**: CM-셀룰로오스 정제 후 각 α 사슬을 CNBr로 절단. 생성된 펩타이드를 CM-셀룰로오스 크로마토그래피로 분석하여 콜라겐 유형 동정.
7. **분자체 크로마토그래피**: 아가로스(Agarose A-5m) 컬럼으로 콜라겐 사슬의 분자량 확인.

## Results

- 초기 연골세포 집단("floaters")에서는 α1(II) 사슬만 검출됨(α1:α2 비율 > 50). 이는 Type II 콜라겐[α1(II)]₃임을 의미.
- 계대가 진행될수록 α1:α2 비율이 점진적으로 감소: 1차 계대 26.4 → 2차 8.4 → 3차 5.5 → 4차(최종) 2.9.
- 섬유아세포 클론은 초기부터 α1:α2 비율 약 2.0으로 Type I 콜라겐만 지속적 합성.
- 최종 계대에서 Type I 콜라겐[α1(I)]₂α2과 Type I trimer[α1(I)]₃가 혼합되어 합성됨. 염 침전 분리 결과: (1) 2.4M NaCl 침전물(α1:α2≈2.0, Type I 콜라겐), (2) 0.01M Na₂HPO₄ 침전물(α1 사슬만, Type I trimer).
- CNBr 절편 분석으로 Type I trimer가 α1(I) 사슬로 구성됨을 확인(α1(II)와는 다른 CNBr 패턴).
- BrdUrd 처리 실험: 1차 계대에서 5일간 BrdUrd 처리 시 α1:α2 비율 3.4로, BrdUrd가 콜라겐 전환(collagen switching)을 가속화함을 확인.
- 추가 실험: 투석된 배아 추출물(dialyzed embryo extract) 배양에서도 세포가 편평해지지 않고도 콜라겐 전환이 유도됨 → 콜라겐 전환이 단순히 세포 노화에만 의존하지 않음을 시사.

## Perspective

- 본 연구는 닭 연골세포 표현형이 in vitro에서 불안정함을 콜라겐 유형 전환(collagen switching)을 통해 증명. 초기 연구의 섬유아세포 과증식 해석을 배제하고 연골세포 자체의 탈분화임을 확인.
- 연골세포가 배양 중 Type II 콜라겐에서 Type I + Type I trimer로 전환되는 현상은 BrdUrd 배양에서도 동일하게 관찰 → BrdUrd는 세포 노화 과정을 가속화하는 역할.
- Type I trimer의 합성이 섬유아세포에서는 명확히 관찰되지 않아, 연골세포 집단에 상대적으로 특이적인 현상일 가능성.
- in vivo 노화 과정에서 유리질 연골이 Type I 콜라겐을 합성할 수 있음을 예측. 골관절염 연골에서 Type I 콜라겐 검출 보고는 이와 일치.
- "Collagen switching"은 세포 노화 이외의 요인(배아 추출물 배양에서도 관찰됨)에 의해서도 유도될 수 있어, BrdUrd 삽입에 의한 무작위 오류 축적(error accumulation) 기전 제안.
- 연골세포의 제한적 수명(약 30 population doublings)은 Hayflick 한계와 일치하며, 세포 노화 연구에 중요한 모델을 제공.
